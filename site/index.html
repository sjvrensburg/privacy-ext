The personal bits never leave your machine.

ClipCloak watches for text you paste into web forms, finds the names, emails, phone and ID numbers inside it with a model that runs on your own computer, and hands back a redacted copy. You decide what actually gets pasted.

A thin Chrome and Firefox extension talks to a local daemon. The model is the work; the extension is just the paste hook and the choice.

How a paste is handled

01

The paste is caught

A content script intercepts the paste in any editable field before the text reaches the page, so nothing is committed until you have seen the redacted version.

02

A local model reads it

The text is sent to a daemon bound to localhost, which runs the GLiNER2 model on device and returns the spans it found along with a masked copy. Long pastes are scanned in overlapping windows.

03

You choose what lands

A small prompt offers the redacted text or the original. Pick one and it is inserted. The daemon keeps nothing, and no request ever leaves the machine.

What ClipCloak looks for

The model is schema driven, so the label set is configurable. Out of the box it targets the identifiers that most often slip into a support ticket or a chat box. The fine tuned build adds South African name coverage the base model missed.

Detection runs at a threshold you set in the popup, near 0.55, trading recall against false positives.

Nothing leaves your machine

The model runs in a daemon that binds to 127.0.0.1 only. It is not reachable from the network. A bearer token, generated for you by the desktop app, gates every request, and the daemon answers cross origin calls only for the pinned extension.

There are no accounts, no telemetry, and no cloud. The code is MIT licensed and the model weights are the ones named in the repository, so you can verify every claim on this page.

Confirm the daemon is up and answering, on the loopback address, with one request:

# the extension talks to this; so can you
curl http://127.0.0.1:8731/health

# classify a line the way the extension does
curl -s http://127.0.0.1:8731/classify \
  -H "Authorization: Bearer $PII_TOKEN" \
  -d '{"text":"Call me on 082 555 0117"}'

Install it

ClipCloak has two halves: the engine, a small Rust daemon that holds the model, and the extension, which hooks pastes in your browser. The desktop app bundles the engine and pairs the extension for you with no token to type.

Prefer to run it yourself? Build the daemon and load the extension unpacked. Full notes, including the Firefox native messaging setup, are in the repository.

Releases Setup notes

Run the engine, then load the browser half:

# 1. build and start the daemon
cd server && cargo build --release
PII_TOKEN=your-secret ./run.sh

# 2. Chrome or Edge: chrome://extensions
#    enable Developer mode, Load unpacked,
#    pick extension-client/

# 3. Firefox: about:debugging, This Firefox,
#    Load Temporary Add-on, extension-firefox/

With the desktop app running, steps 1 and the token are handled for you.
